# Об'єднання Фінансів і Гаманців у єдиний потік

## Що зміниться для користувача

**Менеджер / незалежний репетитор:**

- На сторінці **Фінанси** з'являється головна CTA-кнопка **«+ Зафіксувати оплату»** (поруч із фільтрами). Вона відкриває bottom-sheet з двома варіантами:
  - **💸 За конкретний урок** — список неоплачених уроків учня з тапом «Оплачено»
  - **📦 Передоплата / пакет** — форма гаманця прямо тут (учень + репетитор + уроки/сума + коментар)
- У стрічці транзакцій передоплати показуються поряд з уроками, відсортовані хронологічно: рядок виду `💼 Передоплата · Учень ↔ Репетитор · +1800 ₴ · 3 уроки`
- У фільтрах додається четвертий — **Тип: Уроки / Передоплати / Всі**
- У рядку учня поряд з іменем — мікро-бейдж балансу гаманця (👛 2 уроки зеленим / ⚠️ 0 помаранчевим). Тап по бейджу → відкриває діалог гаманця цієї пари
- Пункт меню/підрозділ профілю **«Гаманці»** перейменовано на **«Передоплати»** (заголовок сторінки теж). Зі сторінки Фінансів додається shortcut «Усі передоплати →»

**Доступ:** менеджер бачить усі пари; незалежний репетитор — лише своїх учнів (поточні RLS це вже гарантують через `wallet_topup` / `student_wallet_balances`).
