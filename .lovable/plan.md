## Рішення №1: Реєстрація по телефону — залишаємо як є

**Email обовʼязковий, телефон опціональний** (поточна схема). Чому:

| Критерій | Email-only | Phone-only (SMS) |
|---|---|---|
| Вартість | $0 (Lovable Cloud) | $0.04–0.30 за SMS, потрібен Twilio |
| Закордон | Працює всюди | Потрібен sender ID для деяких країн |
| Recovery | Легко | Якщо змінив номер — акаунт втрачено |
| Інвайт-флоу | Уже реалізовано | Треба окрема логіка |
| Notifications | Email + Telegram уже є | Дублює існуюче |

**Що залишається:** телефон як опціональне поле в профілі (для контакту тьютор↔учень і для ghost-merge по телефону, якщо репетитор додав так).

**Майбутнє (якщо захочеш login без паролю):** passwordless OTP через ваш Telegram-бот, а не SMS. Нуль витрат, глобально. Це окрема задача — не зараз.
